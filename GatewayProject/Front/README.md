# Frontend (Vite + React + TypeScript + Tailwind)

Pequeno frontend para testar as APIs via Gateway.

Requisitos:
- Node.js (>=18 recomendado)
- npm ou pnpm

Instalação e execução:

```powershell
cd GatewayProject\Front
npm install
npm run dev
```

O dev server roda em `http://localhost:3000` por padrão. Certifique-se que o `Gateway` esteja rodando em `http://localhost:5170` (ou ajuste as URLs em `src/pages/*`).
